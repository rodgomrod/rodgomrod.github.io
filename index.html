FORK

La primera practica que realizamos en clase fue hacer un "fork". Hacer un "fork" es hacer una copia de un repositorio ajeno y traernolos a nuestro GitHub. Tambien podemos descargarlo para tenerlo como repositorio local usando el siguiente codigo:

$ git clone https://github.com/USERNAME/data-science-ipython-notebooks
